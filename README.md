# gerador-citacoes
Gerador de citações criado usando tecnologias nativas JS utilizando como premissa consumo de API externa utilizando Fetch API

Este projeto tem como fundamento utilizar um gerador de citações, utilizando conceitos como consumo de API externa com fetch API, CORS e manipulação de dados mockados, seguindo a video-aula do Matheus Battisti na Udemy.

É um projeto  para seguir a lista de 20 projetos do curso do Matheus, segue URL: [https://www.udemy.com/course/20-projetos-de-javascript-aprenda-html-css-e-javascript/]
